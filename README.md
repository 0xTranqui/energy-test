# Flexible

Create your own DAO interface.

## Running Flexible Locally

First, clone the repository:

```bash
$ git clone https://github.com/public-assembly/flexible.git
```

Second, install dependencies:

```bash
$ pnpm install
```

Lastly, run the development server:

```bash
$ pnpm dev
```
